eBay Price Benchmarks

FilmDB can show median sold prices for expired film on eBay listings. This requires a free eBay Developer App ID (OAuth Application Token).

How to get one:
  1. Register at developer.ebay.com
  2. Create a new application (Production keyset)
  3. Generate a Client Credentials OAuth token
  4. Paste the token below
This token is stored locally in your browser and never sent anywhere except to the eBay API.

About

FilmDB overlays film stock data on B&H Photo, Adorama, Film Photography Store, and eBay.

Film metadata is bundled locally — no tracking, no account required.
Sample scans link to Analog.cafe. Development recipes link to FilmDev.org.

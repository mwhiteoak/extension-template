## Notes on eBay App ID Feature

The expired-film price benchmark feature requires the user to configure a free eBay Developer App ID in the extension options. This should be mentioned in the Privacy practices section as "optional network access using the user's own API credentials." The extension makes no eBay API calls if no App ID is set.

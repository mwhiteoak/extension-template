# Thread Scout — DMC Cross-Stitch Stash Tracker

**Stop buying threads you already own.**

Thread Scout lives in your browser and watches for DMC color codes on every pattern page. When it spots one, it instantly shows whether you own that thread or still need it — right next to the number, without any clicking or tab-switching.

---

## Know your stash at a glance

While browsing Etsy cross-stitch listings, DMC.com, or 123stitch.com, Thread Scout automatically injects **Owned** and **Need** badges next to every DMC color code it detects. Hover any badge for the full color name, hex swatch, and equivalent codes from Anchor and Madeira.

---

## Features

### Owned / Need Badges
- Detects DMC color references in any format: "DMC 310", "DMC #321", "DMC310", "White", "Ecru", "B5200"
- **Green "Owned"** badge — you have this thread in your stash
- **Amber "Need"** badge — this one is missing from your collection
- Hover tooltip shows: color name, hex swatch, Anchor equivalent, Madeira equivalent

### Etsy Quick-Add Bar
- On Etsy listing pages, a summary bar appears listing all the threads you still need
- One-click search: opens an Etsy search for each missing DMC number in a new tab
- Threads are staggered to avoid pop-up blockers

### Stash Manager Popup
- **My Stash tab** — browse and search all threads in your collection; remove any thread instantly
- **Add Thread tab** — type a DMC number or name to get an autocomplete preview with color swatch, then add to stash
- **Import CSV tab** — drag-and-drop (or browse) a CSV file of DMC numbers; supports one-per-line, comma-separated, or header rows

### Works offline
All DMC color data and cross-brand equivalents are bundled locally with the extension. No account required, no external API calls, no data ever leaves your device.

---

## Supported Sites

- **Etsy** (etsy.com) — all cross-stitch pattern and supply listings
- **DMC.com** — official DMC product pages
- **123stitch.com** — pattern and thread product pages

---

## How It Works

1. Install Thread Scout — no sign-up required
2. Open any Etsy, DMC.com, or 123stitch listing that lists DMC thread numbers
3. Owned/Need badges appear automatically next to every detected code
4. Use the popup to manage your stash: add threads manually or import a CSV from your existing catalog

---

## Privacy

Thread Scout stores your stash in `chrome.storage.local` — it never leaves your browser. The extension makes no network requests; all color data and equivalents are bundled locally.

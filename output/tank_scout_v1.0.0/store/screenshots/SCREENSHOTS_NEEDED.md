All screenshots must be 1280×800 px (Chrome Web Store requirement).

Load Tank Scout unpacked, navigate to a LiveAquaria fish listing (e.g., a Neon Tetra or Betta page). Capture the injected compatibility panel showing:

- Tank size, pH, temperature stats

## How to capture

1. Go to `chrome://extensions/`
2. Enable Developer Mode
3. Click "Load unpacked" → select `output/tank_scout_v1.0.0/`
4. Navigate to the target pages and interact as described above
5. Use DevTools device toolbar or browser zoom to achieve 1280×800
6. Screenshot with OS tool or DevTools screenshot

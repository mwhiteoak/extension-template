# D&D Beyond DM Prep Companion

**Run smoother sessions — never leave D&D Beyond again.**

Dungeon Masters who use D&D Beyond waste precious table time clicking between tabs to track initiative, look up monster stats, and estimate encounter difficulty. The DM Companion sidebar keeps everything a hotkey away.

---

## Free Features

### ⚔ Initiative Tracker
- Add any number of combatants (players and monsters) with name, initiative roll, HP, and AC
- Combatants automatically sort by initiative
- Active turn is highlighted — click **Next ▶** to advance the round
- Track damage and healing with a quick ± button on each combatant
- Move combatants up or down in turn order at any time
- Round counter auto-increments when the turn order wraps

### 📝 Session Notes
- Freeform scratch pad synced to your Chrome account via `chrome.storage.sync`
- Available on every device where you're signed into Chrome
- Auto-saves as you type — no manual save required

---

## Pro Features ($6/month)

### 📊 Encounter Difficulty Estimator
- Input party size and average level → instantly see Easy / Medium / Hard / Deadly XP thresholds
- Uses official D&D 5e DMG XP tables (levels 1–20)
- Add monsters from the SRD search to the encounter, and the adjusted XP (with encounter multiplier) is calculated in real time

### 🐉 Monster Stat Quick-Reference
- Search any creature from the D&D 5e Systems Reference Document without leaving the page
- Displays AC, HP, speed, all six ability scores with modifiers, and key combat notes
- One click to add any monster to the initiative tracker (initiative rolls automatically using Dexterity modifier)

### ⚔ Encounter Builder
- Build an encounter by searching and adding SRD monsters with quantity controls
- Send the whole encounter to the initiative tracker with a single click — initiative rolls included
- Drag count up or down per monster type before sending

---

## How It Works

- Open any D&D Beyond page (encounter, character, compendium)
- Press **Shift+D** to open the sidebar — or click the ⚔ icon in your Chrome toolbar
- The sidebar uses Shadow DOM so it never conflicts with D&D Beyond's styles

---

## Privacy

- No account required
- No data leaves your browser — all state is stored locally or in your Chrome sync storage
- No external API calls for monster data — the SRD is bundled directly in the extension

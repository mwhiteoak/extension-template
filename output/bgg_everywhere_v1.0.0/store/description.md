# BGG Everywhere — BoardGameGeek Companion

Board game shoppers — stop switching tabs. BGG Everywhere overlays BoardGameGeek data directly on Amazon, Target, Walmart, Miniature Market, and other board game retailers.

## What you see on every product page

- ⭐ BGG community rating and total vote count
- 🏆 Overall board game rank + category rank (e.g. #47 Strategy)
- 👥 Player count with community "best at" recommendation
- ⏱ Play time (publisher range)
- 🧠 Complexity score (1–5 weight scale)
- 🔗 One-click link to the full BGG listing

## Supported Sites

- Amazon
- Target
- Walmart
- Miniature Market
- Cool Stuff Inc.
- Game Nerdz
- Fun Again Games

## Pro ($5/mo)

- Add to Want to Buy / Want to Play lists in one click
- Collection check — see if you already own a game before buying

## How it works

BGG Everywhere detects the board game on the current product page, looks it up on BoardGameGeek's public XML API, and shows the data in a floating panel. Results are cached for 24 hours for instant re-loads.
